# Story 5.2: Validate Work Against AGENTS.md

Status: done

## Story

As an Autonomous Project Maintainer,
I want work checked against `AGENTS.md`,
so that implementation does not violate AI Constitution rules.

## Acceptance Criteria

1. Given `AGENTS.md` and task output exist, when constitution validation runs, then violations are reported with rule reference.
2. Given unresolved critical violations exist, then task cannot be marked done.
3. Given approved exceptions exist, then approved exceptions are written to Journal.
